分页器（Pagination）

示例

基本形式

<u-pagination :total="11"></u-pagination>
<u-pagination :total="11" :current="6"></u-pagination>

显示数目

使用side和around属性可以改变显示的数目。

<u-pagination :total="11" :current="6" :side="1" :around="3"></u-pagination>

禁用

<u-pagination :total="11" disabled></u-pagination>

API

Class

Pagination

Extends Base。

Options

new Pagination()

ParamTypeDefaultDescription
props.current number 1 Current page number
props.total number 11 Total count of pages
props.side number 2 Count of pages at side position
props.around number 5 Count of pages around the current position
props.readonly boolean false Readonly
props.disabled boolean false Disabled

Methods

pagination.select(page)

Select a page

ParamTypeDefaultDescription
page number The page number to select
ReturnTypeDescription
No Returns

Events

pagination.$on('select')

Emit when selecting a page

PropertyTypeDescription
current number The current page
