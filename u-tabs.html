API

Class

Tabs

Extends Base。

Options

new Tabs()

ParamTypeDefaultDescription
props.selectedIndex number 0 Index of selected tab
props.readonly boolean false Readonly
props.disabled boolean false Disabled

Methods

tabs.select(index)

Select a tab

ParamTypeDefaultDescription
index number Index of tab to select
ReturnTypeDescription
No Returns

Events

tabs.$on('select')

Emit when selecting a tab

PropertyTypeDescription
selectedTab number The selected tab
selectedIndex number Index of selected tab
